# **정산해욥(JeongSanHaeYOP)**


OCR로 출금 내역을 스캔하고 원하는 내역만 계산할 수 있도록 하는 앱 서비스


## Features

- Main Feature: OCR로 출금 내역을 인식하고 체크박스로 선택
- Additional Feature: 정산 내역을 공유
